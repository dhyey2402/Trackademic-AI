# EduCore AI System
Full Stack B.Tech Curriculum ERP System

## Backend
Built with Python, Flask, SQLite, and Anthropic API.
1. `cd backend`
2. `python -m venv venv`
3. `venv\Scripts\activate` (Windows)
4. `pip install -r requirements.txt`
5. Create `.env` and add: `ANTHROPIC_API_KEY=your_key`
6. `python app.py`

## Frontend
Built with React, Vite, and Tailwind CSS.
1. Install Node.js if you haven't.
2. `cd frontend`
3. `npm install`
4. `npm run dev`

Navigate to `localhost:5173/generator` to use the AI application.
